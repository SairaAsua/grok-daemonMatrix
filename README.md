# DaemonCraft

DaemonCraft es un videojuego en Minecraft donde cada jugador tiene un agente de inteligencia artificial como compañero de vida.

El agente sabe quién sos, te ayuda a crear historias, genera mundos como un holodeck, y crece con vos desde el primer día.
